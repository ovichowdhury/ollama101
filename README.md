# ollama101

```sh
python -m venv <ai-env>
```

For activating the environment:

- On Linux:

  ```sh
  source <ai-env>/bin/activate
  ```

- On Windows:
  ```sh
  <ai-env>\Scripts\activate.bat
  ```

Freezing packages:

```sh
pip freeze > requirements.txt
```

Installing packages from the file:

```sh
pip install -r requirements.txt
```
